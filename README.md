# 🛒 End-to-End E-Commerce Analytics and Machine Learning Project

An end-to-end data analytics and machine learning project built using the Brazilian E-Commerce Public Dataset by Olist. This project covers the complete workflow from SQL analysis and exploratory data analysis to interactive Power BI dashboards and machine learning models.

---

## 📌 Project Overview

The objective of this project is to analyze e-commerce data and extract meaningful business insights related to customers, products, revenue, and logistics. In addition, machine learning techniques were used to perform customer segmentation and churn prediction.
